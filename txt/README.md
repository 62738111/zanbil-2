## راهنما
- `*` **پوشه**   _(در این قسمت باید نام پوشه نوشته شود)_
- `~` **صحفه دلخواه**   _(در این قسمت نام خود فایل را باید بنویسید)_
- `/` **آندرلاین**   _(برای رفتن به مرحله بعدی)_
- `&` **(شاخه متغیر)***_به عنوان نام شاخه شما_
